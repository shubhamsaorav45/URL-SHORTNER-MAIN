<br/>

The server is written in Java using the Spring Boot framework. For database layer Mysql is used with Hibernate and Spring Data JPA. The frontend is built using ReactJS.

### Back-End

```bash
  mvn spring-boot:run
```

### Front-End

```bash
  npm start
```
